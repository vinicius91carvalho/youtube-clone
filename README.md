# Youtube Clone

A project from neetcode.io course

# Features

* List videos
* Watch a video
* Sign in/out
* Upload a video
* Watch the transcoded video

* TypeScript
* Next.js
* Express.js
* Docker
* FFmpeg
* Firebase Auth
* Firebase Functions
* Firebase Firestore
* Google Cloud Storage
* Google Cloud Pub/Sub
* Google Cloud Run
